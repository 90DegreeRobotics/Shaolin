# Mandarin Calligraphy Journal Method

This method exists because ordinary journaling can become shapeless. The page needs a governor.

The sequence is:

```text
English thought -> core meaning -> Mandarin phrase -> pinyin -> character -> hand practice
```

English records the day. Mandarin compresses the meaning. Calligraphy slows the body down until the thought has weight. Pinyin must include tone marks or tone numbers; tones are not decoration.

## Nightly Page Template

```text
Date:
Shaolin day:

Physical truth:
Emotional truth:
Conduct truth:

One sentence worth keeping:
Core word / phrase:

Mandarin:
Pinyin:
Character focus:

Write the character or phrase 9 times:

What this character asked of me:
Tomorrow's vow:
```

## Rules

- One page is enough.
- One honest sentence is enough.
- One character is enough.
- Verify translations before treating them as settled.
- Prefer simplified characters first unless a source explicitly teaches traditional.
- Use correct stroke order when possible.
- Do not use Mandarin as decoration. If the character goes on the page, it must carry lived meaning.

## First Example

```text
English thought:
I opened the gate so knowledge may help another person walk a better path.

Core phrase:
Knowledge illuminates the path.

Mandarin:
知识照亮道路

Pinyin:
zhī shi zhào liàng dào lù

Character focus:
道
dào
path, way, road, principle
```

Write 道 9 times.

## Starter Character Set

Use these early. They are simple enough to return to, but deep enough to matter.

| Character | Pinyin | Working Meaning | Journal Question |
| --- | --- | --- | --- |
| 道 | dào | path, way, principle | Did I walk the path or only imagine it? |
| 静 | jìng | stillness, quiet | Where did I become quiet instead of reactive? |
| 仁 | rén | humaneness, benevolence | Where did I make life easier for another person? |
| 忍 | rěn | endurance, restraint | Where did I endure without becoming cruel? |
| 明 | míng | brightness, clarity | What became clearer today? |
| 行 | xíng | action, conduct, walking | What did I actually do? |
| 心 | xīn | heart, mind | What ruled my heart-mind today? |
| 身 | shēn | body, self | What did the body report? |
| 气 | qì | breath, vital process | How was my breath today? |
| 和 | hé | harmony, peace | Where did I create or disturb harmony? |

## Weekly Review

At the end of each week, choose one character that best represented the week. Write:

```text
This week's character:
Why:
Where I lived it:
Where I failed it:
Next week's correction:
```

This turns vocabulary into conduct.
