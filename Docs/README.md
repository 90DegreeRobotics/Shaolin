# Docs
## Supplementary Study Shelf

This folder holds reference material that supports the main path without
replacing it.

Authority order remains:

1. `README.md`
2. `STATUS.md`
3. `SAFETY.md`
4. `1yeartoShaolin.md`
5. `TRAINING_HALL.md`
6. lane documents under `Docs/`, `Mandarin/`, `Diet/`, `Wisdom/`, and `Forms/`

The main manual is still the training container. The study documents here are
used to clarify techniques, organize outside resources, and give Chirox more
real text to retrieve from. They do not create lineage authority, medical
advice, or permission to skip the safety rules.
