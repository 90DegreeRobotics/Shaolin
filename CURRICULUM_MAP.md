# Curriculum Map
## One Path, Multiple Lanes

This map keeps the repo usable. The manuals are deep; the daily action must be
simple.

## Day 1 Minimum

Do this before adding anything:

1. Sit for 3 minutes.
2. Hold one careful stance.
3. Walk without the phone.
4. Write three honest lines.

That is a valid day.

## First 30 Days

| Lane | Practice | Chirox role |
| --- | --- | --- |
| Sit | 3 to 10 minutes breath meditation | record only |
| Stand | Horse stance, parallel ready, wuji standing | measurable if framed |
| Move | walking, gentle mobility, first Eight Brocades movement | record and range-measure |
| Record | daily check-in | template and sealed record |
| Study | manual, safety, beginner guide | read/retrieve |
| Mandarin | tones, basic journaling pipeline | template and study lane |

Do not collect techniques in the first 30 days. Build return.

## Quarter 1

Goal: stabilize the floor.

- daily invariant
- safety rule internalized
- Horse stance starts shallow
- Eight Brocades begins gently
- food and sleep stabilized enough to support practice
- Chirox used for record, reading, and simple stance measurement

## Quarter 2

Goal: build the training hall.

- add Bow stance, Empty stance, Crane/one-leg balance
- add more conditioning holds and counted reps
- add simple load and grip work
- staff may appear as shoulder opener and slow grip practice
- no fast weapon work

## Quarter 3

Goal: increase capacity without losing honesty.

- longer sessions
- more forms recorded on the visual timeline
- impact work stays light and gradual
- staff basics may become a regular record-only lane
- outdoor practice can expand if lawful and quiet

## Quarter 4

Goal: consolidate.

- compare monthly trends
- reduce novelty
- review footage
- prepare the next year's training questions
- seek qualified human correction if possible

## Chirox Classification

| State | Meaning | Examples |
| --- | --- | --- |
| Measurable | Deterministic geometry exists and Chirox may speak limited corrections. | Horse, Bow, Crane, Plank, Wall Sit, counted reps. |
| Recordable | Chirox may film/log/range-measure but not grade correctness. | Eight Brocades, Wu Bu Quan, staff basics, kicks, flowing forms. |
| Deferred | Too risky, too advanced, or too noisy for current solo grading. | Tornado kick, deep sweeps, fast weapon forms, iron body claims. |

## Chirox Commands

```powershell
python -m chirox.cli library
python -m chirox.cli today
python -m chirox.cli log daily --print-template
python -m chirox.cli train --stances horse,bow,empty_stance --seconds 30
python -m chirox.cli record --exercise wu_bu_quan --seconds 120 --no-show
```

Use the commands as proof helpers, not as a substitute for practice.
