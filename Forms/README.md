# Forms — The Printable Practice

Paper forms for the daily and weekly rituals. The markdown files are the
canonical sources; the `print/` folder holds self-contained HTML versions
styled like GitHub's rendering, tuned for paper (larger type, ruled lines,
a character grid for calligraphy).

## The Forms

- [DAILY_FORM.md](DAILY_FORM.md) / `print/daily_form.html` — one sheet per
  day, front and back: the bells checklist and Daily Check-In on page 1,
  the Mandarin journal page with a 9-box character grid on page 2.
  Print a stack; fill one every night at the Evening Gate.
- [WEEKLY_REVIEW_FORM.md](WEEKLY_REVIEW_FORM.md) / `print/weekly_review.html`
  — the Sunday Weekly Dojo Review: the review questions, the six-pillar
  touch check, and the week's character.

## How to Print

1. Open the HTML file in any browser (double-click).
2. Ctrl+P. Set margins to Default, disable headers/footers, enable
   two-sided printing for the daily form.
3. Print a week's stack at a time: 7 daily sheets + 1 weekly sheet.

The layouts are Letter-sized with print margins built in. The completed
paper page is the source of truth for the night; the Dojo Record entry
(`chirox log`) is its sealed twin.
